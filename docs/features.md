# Features

Here's what's included in this setup. It's built on Hyprland and Quickshell, and includes a bunch of stuff to make it actually usable as a desktop environment.

## Core Components

### Window Manager

- **Hyprland** - Tiling Wayland compositor with animations and window management
- **Hardware acceleration** - GPU acceleration is enabled
- **Window rules** - You can customize how windows behave

### Shell System

- **Quickshell** - Qt-based shell with widgets, launchers, and desktop stuff
- **Modular** - Everything is split into components so you can customize it

## Desktop Components

### Top Bar

The top bar shows system info and gives you quick access to stuff:

- Time and date
- Network status
- Audio controls
- Battery (if you're on a laptop)
- Notifications
- Quick settings

### Dock

Application dock at the bottom:

- App shortcuts
- Shows which apps are running
- Window previews
- Animations
- You can customize how it looks

### Control Center

System control panel with:

- Audio mixer
- Display settings
- Network stuff
- Bluetooth
- System info
- Power management

### Application Launcher

You've got a few launcher options:

- **Quickshell Spotlight** - Main launcher with search
- **App Drawer** - Grid view of all your apps
- **Fuzzel** - Fallback launcher (dmenu-style)
- **Anyrun** - Another fallback option

### Overview

Workspace overview shows:

- All your open windows
- Workspace management
- Quick window switching
- Organize your workspaces

### Dark Dash

Customizable dashboard with:

- System widgets
- Quick actions
- Info display
- You can arrange it however you want
